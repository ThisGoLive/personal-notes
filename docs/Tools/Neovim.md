## 3 快捷键

### 3.1 vim 快捷键

### 3.2 lazyvim 快捷键

:LazyVimExtras 安装

| 命令                  | 描述               |
| --------------------- | ------------------ |
| ctr + /               | 打开 tem           |
| leader bd             | 关闭 tab 页 buffer |
| leader w-             | 下添加窗口         |
| leader w\|            | 右边添加窗口       |
| shift h l             | tab 页移动         |
| ctr h l               | tree tab 移动      |
| shift h               | tree 显示隐藏      |
| shift k               | 查看定义           |
| leader sm or ` '      | 跳转到标记         |
| m +                   | 添加标记           |
| :delm! \| delm A-Z0-9 | 清除标记           |
