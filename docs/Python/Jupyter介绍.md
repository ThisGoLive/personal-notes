- 启动位置位，为 当前命令 行打开的路径，并且与 配置相同 jupyter --runtime-dir 查看运行得到路径 jupyter --paths 各个路径 C:\Users\GL\AppData\Roaming\jupyter 路径下为 配置路径  "notebook_dir": "C:\\Users\\GL", 为 运行后服务 打开的路径，显示该路径下全部的文件等

Untitled.ipynb 文件

- 创建
- 文件位置
