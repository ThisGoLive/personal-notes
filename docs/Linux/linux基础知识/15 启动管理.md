#### 默认运行级别：

+ 0 关机
+ 1 单用户模式  启动基本服务，用于修复系统
+ 2 命令行模式  没有nfs服务
+ 3 完全模式
+ 4 系统保留
+ 5 图像模式
+ 6 重启
