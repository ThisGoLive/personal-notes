# 第三讲 给Linux初学者的建议 #

## 3.1 给初学者的建议-注意事项 ##
	/dev 下管理的硬件 所以的全是以文件保存的
**不是以扩展名来区分类型**

1. 压缩包：.gz .bz2 .tar.bz2  .tgz
2. 二进制软件包： .rpm
3. 网页文件： .html .php
4. 脚本文件： .sh
5. 配置文件： .conf

但是系统不需要识别，而是给管理员看的

**所有的存储设备都必须挂载后才能运行使用，如硬盘，U盘**
