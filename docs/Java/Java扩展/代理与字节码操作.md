# 代理与字节码操作

2024年2月20日

## 字节码操作

以编程方式操作字节码的库

+ ASM - 通用的低级字节码操作和分析。
+ Byte Buddy - 使用流畅的 API 进一步简化字节码生成。
+ bytecode-viewer - Java 8 Jar & Android APK 逆向工程套件。 （仅限 GPL-3.0）
+ Byteman - 在运行时通过 DSL（规则）操作字节码； 主要用于测试/故障排除。 （LGPL-2.1 或更高版本）
+ cglib - 字节码生成库。
+ Javassist - 尝试简化字节码编辑。
+ Mixin - 使用真正的 Java 代码在运行时操作字节码。
+ Perses - 根据混沌工程原理在字节码级别动态注入故障/延迟。
