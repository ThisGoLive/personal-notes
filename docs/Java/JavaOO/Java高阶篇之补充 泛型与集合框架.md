## 一、概念

在jdk1.5中推出的。目的是简历具有类型安全的集合框架。

创建类时<> 中泛型。可以是任何复杂数据类型。在该类中可以拥有这个泛型的引用。

```java
class Cone<E>{
 double height;
 E e;
    public Cone (E e){
        this.e = e;
    }
}
```

基本就这样，很基础。
